PM Jobsheet 3 – Percabangan, Perulangan, dan Git Branching pada Dart
Pada jobsheet ini dimulai dari pembuatan repository baru di github dan menghubungkan dengan jobsheet yang dibuat pada VSCode. program ini menggunakan perilangan while dan percabangan if-else.
perulangan digunakan agar program terus meminta input dari pengguna sampai tebakan yang dimasukkan sama dengan angka yang telah ditentukan. Sedangkan percabangan digunakan untuk menentukan apakah tebakan pengguna benar atau salah.

Setelah program dasar selesai dibuat, proses pengembangan dilanjutkan menggunakan Git branching. Beberapa branch dibuat untuk mengembangkan fitur secara terpisah agar perubahan pada program dapat dikelola dengan lebih rapi dan terstruktur.

- Branch 
dalam jobsheet ini terdapat beberapa branch selain main untuk mengembangkan fitur secara terpisah. setiap branch digunakna untuk menambahkan atau memperbaiki bagian tertentu dari program. Branch yang digunakna untuk :
- menambahkan fitur input dari pengguna,
- menambahkan logika permainan tebak angka,
- memperbaiki pesan output agar lebih informatif.
setelah fitur pada setiap branch selesai dikembangkan, perubahan tersebut kemudian digabungkan kembali ke branch utama menggunakan proses merge.

- Implementasi Program
pada program tebak angka ini dibuat dengan menentukan sebuah angka yang harus ditebak oleh pengguna. program ini akan meminta pengguna memasukkan angka melalui input dari keyboard. lalu program akan melihat apakah angka tersebut sama, jika angka yang di masukkan tidak sama dengan angka yang telah di tentukan, maka program akan menampilkan bahwa tebakan salah. 
ketika pengguna berhasil menebak angka yang benar, program akan menampilkan pesan bahwa tebakan tersebut benar dan perulangan akan berhenti.
